Installation

To install Subsonic, double-click the Subsonic Installer and follow the on-screen instructions. The Installer will tell you when the installation process is complete. Subsonic is installed as a Windows service, and is started automatically when the computer starts. Double-click the Subsonic icon in the Windows System Tray to launch the Subsonic control panel.

By default, Subsonic is accessible on http://localhost/, but the port number and URL can be changed from the control panel.

If you are upgrading an existing installation of Subsonic, your previous settings (stored in c:\subsonic) will be preserved.
